# Swaggergen

Swaggergen is a tool to reverse engineer open-api specification from the `proxify` stored logs.


# Installation

Download the ready to run [binary](https://github.com/projectdiscovery/proxify/releases/) or install/build using GO

```shell
go install -v github.com/projectdiscovery/proxify/cmd/swaggergen@latest
```

# Usage

```shell
swaggergen -h
```

This will display help for the tool. Here are all the switches it supports.

```shell
Usage:
  ./swaggergen [flags]

Flags:
   -log-dir string           proxify output log directory
   -api, -api-host string    api host (example: api.example.com)
   -os, -output-spec string  file to store openapi/swagger spec(example: swagger.yaml)
```

### Running Swagggergen

Runs swaggergen on the `logs` generated by proxify, generates a swagger spec file for all the requests made to `localhost:8080`, and store it in `open-api.yaml`:
```shell
swaggergen -api localhost:8080 -log-dir ./logs -os open-api.yaml
```
